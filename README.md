# AICOS

Initial project skeleton for AICOS.

## Prerequisites

- Python 3.12
- [uv](https://docs.astral.sh/uv/)

## Setup

The project targets Python 3.12. A local virtual environment is created with:

```bash
uv venv --python 3.12
```

There are intentionally no dependencies, frameworks, application modules, or agent implementations yet.

## Layout

```text
backend/   Backend implementation (reserved)
frontend/  Frontend implementation (reserved)
config/    Configuration files (reserved)
data/      Local data directories (reserved)
docs/      Project documentation
tests/     Automated tests (reserved)
```
